Smart Question Paper Generator

Smart Question Paper Generator is a Django web application designed to create, manage, and generate examination question papers efficiently. Teachers can add questions to a question bank, filter by subject/year/unit, assemble papers with section-wise marks, validate totals, and export the final paper as a PDF.

Key Features:

Question Bank CRUD: Add, view, filter, and delete questions by subject, year, or unit. Flexible Paper Generation: Configure sections, question counts, and marks per question. Validation Logic: Ensures total marks equal the requested maximum before allowing PDF generation. PDF Export: Generates a professional PDF of the final question paper. User Interface: Clean, responsive UI built with Bootstrap. Interactive Feedback: Uses session messages and animated notifications for user actions.
